# AI探究导师 - 酵母菌呼吸作用实验

一个集成AI的交互式科学探究实验模拟器，使用苏格拉底式提问法引导学生完成酵母菌呼吸作用的探究实验。

## 功能特点

- 🤖 **AI引导**：集成DeepSeek API，苏格拉底式提问引导学生深入思考
- 🧪 **实验模拟**：可视化实验装置，包含量筒、气球膨胀、气泡动画
- 🖱️ **拖拽组装**：第一组装置手动拖拽组装，体验实验操作流程
- 📊 **数据记录**：实时记录和精确验证实验数据（保留2位小数）
- 🎛️ **参数控制**：5组温度滑块，支持设置不同温度对照实验
- ⏱️ **时间加速**：1秒真实时间 = 1小时模拟时间（共8小时实验）
- 📝 **个性化总结**：AI根据用户探究过程生成个性化课程总结

## 探究流程

### Part 1：问题聚焦
1. **展示现象**：发面场景引入，配合图片展示
2. **问题a**：为什么要加酵母菌？（引导：酵母菌呼吸产生气体）
3. **问题b**：为什么放温暖的地方？（引导：温度影响呼吸速度）
4. **聚焦研究问题**：温度是否影响酵母菌呼吸作用的速度

> 💡 采用苏格拉底式提问法，每个问题最多3轮引导，不直接透露答案

### Part 2：设计实验
5. **材料准备**：酵母菌、水、白砂糖、量筒、气球、皮筋、搅拌棒
6. **组装装置**：第一组手动拖拽组装，其余4组自动完成
7. **识别变量**：
   - 自变量：温度
   - 因变量：气体体积
   - 控制变量：酵母菌/糖/水的用量（多选题）
8. **设置温度**：5组装置设置不同温度（至少2个不同值）
9. **预测结果**：记录用户的预测

### Part 3：实验与结论
10. **运行实验**：8秒模拟8小时发酵过程
11. **记录数据**：填写5组气体体积数据（精确匹配）
12. **分析数据**：AI使用真实实验数据进行分析
13. **对比预测**：显示用户之前的预测，对比实验结果
14. **陈述结论**：温度越高，酵母菌呼吸作用速度越快

### Part 4：课程总结
15. **个性化总结**：回顾用户探究过程中的具体表现
16. **流程图**：SVG可视化展示探究流程
17. **元认知问题**：AI根据探究过程动态生成反思问题

## 气体体积计算公式

```javascript
// 温度 ≥ 25°C：正常发酵
气体体积 = 0.2 × 温度 × 时间 (ml)

// 温度 < 25°C：低温抑制
气体体积 = 0.02 × 温度 × 时间 (ml)
```

## 配置说明

在 `src/config.js` 中配置：

```javascript
export const config = {
  api: {
    baseUrl: 'https://api.deepseek.com/v1/chat/completions',
    apiKey: 'your-api-key-here',
    model: 'deepseek-chat',
  },
  experiment: {
    apparatusCount: 5,           // 实验装置数量
    experimentDuration: 8,        // 模拟实验时长（小时）
    realSecondsPerHour: 1,        // 每模拟小时对应的真实秒数
    // ...
  }
};
```
